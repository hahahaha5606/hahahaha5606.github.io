二、CSS

1.CSS的定义

CSS (Cascading Style Sheets) 指层叠样式表(级联样式表、样式表)，CSS是一种标记语言。它有以下作用：

● 设置HTML页面中文本内容

● 图片的外形

● 版面的布局和外观显示样式

2.基础语法

我们把对文本样式进行定义和修饰的CSS代码写在<style></style>中。CSS规则由两个主要部分组成：选择器以及一条或多条声明。

如：选择器{属性名 1 : 属性值 1 ; 属性名 2 : 属性值 2 }

3.CSS选择器

选择器将样式表和标签进行匹配。

3.1.基础选择器

常用基础选择器有4种：

➱ 标签选择器 ：匹配页面中所有的指定标签

➱ id选择器 ：匹配id值唯一的一个标签

➱ 类选择器 ：匹配class属性相同的一组标签

➱ 通配选择器 ：匹配页面中所有标签

标签选择器

标签选择器：是指⽤HTML标签名称作为选择器，按标签名称分类为页面中某⼀类标签指定统⼀的CSS样式。标签选择器能快速为页面中同类型的标签统⼀设置样式，但不能设计差异化样式，只能选择全部的当前标签；

常见的选择标签有：

  • 标题标签（h1~h7）
  • 段落标签（p）、图片标签（img）
  • 布局标签（大盒子 div、小盒子 span）
  • 表格标签（table）
  • 表单标签（form、input、select、textarea）
类选择器

类选择器：如果想要差异化选择不同的标签，单独选⼀个或者某几个标签，则可以使⽤类选择器。这里的“类”指“class属性”。

注意！不能使用标签名字作为类名（如.div）！也不要使用纯数字、中文等命名，尽量使用英文字母表示！

常用：

  • 头：header
  • 内容：content/container
  • 尾：footer
  • 导航：nav
  • 侧栏：sidebar
  • 栏目：column
  • 搜索：search
  • 菜单：menu ...

类选择器-多类名：简单理解就是⼀个标签有多个类class属性。使用方式为：

  • 在标签class属性中写多个类名
  • 多个类名中间必须用空格分开

多类名可以把⼀些标签元素相同的样式(共同的部分)放到⼀个类里面，这些标签都可以调用这个公共的类，然后再调用自己独有的类。

id选择器

id选择器：可以为某id的HTML元素指定特定的样式。CSS中id选择器以"#"来定义。

如果要同时选择并设置多个id样式，有以下两种方法：

1.使用逗号分隔不同id（并集选择器）：#id1,#id2{...}；

2.使用属性选择器：

3.将其设置为同一类（class），然后使用类选择器。

通配符选择器

通配符选择器：使⽤"*****"定义，它表示选取⻚⾯中所有元素(标签)。

我们很少主动使用这个选择器；它的一般使用场景是前段开发避免自己设置的样式与浏览器的自带样式冲突，则使用其事先清除所有元素标签的内外边距。

3.2.复合选择器

复合选择器建立在基础选择器之上，是基础选择器进行组合形成的。复合选择器可以是两个或多个基础选择器，以不同方式组合而成。相较于基础选择器，复合选择器能更加准确高效地选择目标元素。

常用复合选择器为前4种：

➱ （空格）后代选择器 ：选择指定元素后代的所有元素

➱ （>）子选择器 ：选择指定元素子元素的所有元素

➱ （,）并集选择器 ：同时选择多个选择器

➱ （:）伪类选择器 ：用来描述元素的特殊状态（首个元素、被点击元素、鼠标移入元素等）

➱ 交集选择器 ：同时选择复合多个条件的元素。有标签选择器，以标签选择器开头！（如，p.red{}选择类名为red的段落标签）

➱ （+）相邻兄弟选择器 ：选择紧跟在另一个元素后面的元素，且两者有相同父元素

➱ （~）通用兄弟选择器 ：选择指定元素的同级元素的所有元素

➱ （[ ]）属性选择器 ：

  • [属性名] { } -->选择含有指定属性的元素
  • [属性名=属性值] { } -->选择含有指定属性和属性值的元素
  • [属性名^=属性值] { } -->选择以指定属性值开头的元素
  • [属性名$=属性值] { } -->选择以指定属性值结尾的元素
  • [属性名*=属性值] { } -->选择任意部分包含该属性值的元素
  • [属性名~=属性值] { } -->选择任意部分包含该属性值的元素，且该属性值是一个单词（左右空格）
  • [属性名|=属性值] { } -->选择属性值为”属性值“，或以“属性值-”为前缀（- 为连字符，Unicode 编码为 U+002D）开头。典型的应用场景是用来匹配语言简写代码（如 zh-CN、zh-TW 可以用 zh 作为 value）的元素。
  • [属性名 操作符 属性值 i] { } -->在属性选择器的右方括号前添加一个用空格隔开的字母 i（或 I），可以在匹配属性值时忽略大小写（支持 ASCII 字符范围之内的字母）。
  • [属性名 操作符 属性值 s] { } -->在属性选择器的右方括号前添加一个用空格隔开的字母 s（或 S），可以在匹配属性值时区分大小写（支持 ASCII 字符范围之内的字母）。
后代选择器

后代选择器（又称包含选择器）：当标签发生嵌套时，内层标签成为外层标签的后代。后代选择器可以选择父元素里的子元素，注意选择的是子元素！这里的子元素可以是儿子、孙子等。父子之间使用空格隔开。

注：父标签和子标签可以是任意的基础选择器。

子选择器

子选择器（即子标签选择器）：只能选择某标签的最近一级子标签！不能是孙子、重孙等。父子之间用>隔开。

并集选择器

并集选择器：可以为多组标签同时定义相同样式。使用,隔开选择的多个标签，任何形式的选择器都可以作为并集选择器的一部分。

伪类选择器

伪类选择器：用于选择处于特殊状态的元素，通过在元素后加上:状态选取。

伪类选择器可以分为动态伪类、结构伪类、否定伪类、UI伪类、语言伪类五种。

1.动态伪类

常见的动态伪类为：

  • :link 超链接未被访问的状态。
  • :visited 超链接访问过的状态。
  • :hover 鼠标悬停在元素上的状态。
  • :active 元素激活的状态（点击不松）。
  • :focus 获取焦点的元素（当前选中，准备接受用户输入的元素）。

需要注意的是，设置link 、visited 、hover 、active 动态伪类的时候，必须按照link 、visited 、hover 、active 的顺序对操作对象进行设置。且只有表单类元素才能使用:focus 伪类。

2.结构伪类

结构伪类包括：

  • :first-child 所有兄弟元素中的第一个。

  • :last-child 所有兄弟元素中的最后一个。

  • :nth-child(n) 所有兄弟元素中的第 n 个。

  • :first-of-type 所有同类型兄弟元素中的第一个。

  • :last-of-type 所有同类型兄弟元素中的最后一个。

  • :nth-of-type(n) 所有同类型兄弟元素中的 第n个 。

对于n的值：

    1. 0 或不写：什么都选不中 —— 几乎不用。
    2. n ：选中所有子元素 —— 几乎不用。
    3. 1~正无穷的整数 ：选中对应序号的子元素。
    4. 2n 或 even ：选中序号为偶数的子元素。
    5. 2n+1 或 odd ：选中序号为奇数的子元素。
    6. -n+3 ：选中的是前3 个。

其他补充（使用场景更少，了解即可）:

  1. :nth-last-child(n) 所有兄弟元素中的倒数第 n 个。
  2. :nth-last-of-type(n) 所有同类型兄弟元素中的 倒数第n个 。
  3. :only-child 选择没有兄弟的元素（独生子女）。
  4. :only-of-type 选择没有同类型兄弟的元素。
  5. :root 根元素。
  6. :empty 内容为空元素（空格也算内容）。

5.引入方式

CSS样式表分为：行内样式表(行内式) 、 内嵌样式表(嵌入式) 和 外部样式表(链接式) 。

简单来说就是css代码写的位置不同而已，如果标签只出现一次可以考虑使用行内样式写法。若有多组标签且样式需求一致可以使用内嵌样式写法。一般在大的项目css代码很多情况下才使用外部样式写法，也就是将css代码单独写在一个.css文件中，再在html文件中通过<link/>导入即可。

行内样式写法常用于缩放图片等，如：<img src="blogImage/images/image-20250411.png" style="zoom:80%;" />
