一、HTML

1.什么是HTML

HTML是超文本标记语言（Hyper Text Markup Language）是用来描述网页的一种语言。注意，HTML不是编程语言，而是一种标记语言，使用特定的标记（标签）来标识文本中的不同部分或元素，并指示这些部分如何在渲染或处理时被解释或呈现。

简单来说，HTML文件也可以直接称为网页，浏览器的作用就是读取HTML文件，并且以网页的形式去展示它们。

2.标签

HTML标签是由尖括号包围起来的关键词，如<html></html>。HTML标签的类型可分为单标签与双标签：

（1）双标签书写规则：<双标签名称>内容，例如<html>内容</html>；

（2）单标签书写规则：<单标签名称/>，例如<br/>;

3.HTML属性

HTML属性指的是标签属性，HTML标签可以拥有属性，给相关的HTML元素提供更多的信息。

注意：

（1）一个HTML标签可有多个属性；

（2）属性写在HTML元素的开始标签；

（3）属性总是以名称/键值对的形式出现，比如：class="method";

4.HTML字符实体

在 HTML 中，某些字符是预留的，这些字符将被默认为标签。如：在 HTML 中不能使用小于号（<）和大于号（>），这是因为浏览器会误认为它们是标签。如果希望正确地显示预留字符，我们必须在 HTML 源代码中使用字符实体（character entities）:

5.HTML标签骨架

<meta http-equiv="X-UA-Compatible" content="IE=edge">

  • 目的： 告诉 IE 浏览器（Internet Explorer）用最新的渲染模式来解析网页。
  • 解释：
    • http-equiv="X-UA-Compatible"： 表示设置 HTTP 头部信息，这里要设置的是浏览器兼容模式。
    • content="IE=edge"： 指定 IE 浏览器使用 Edge 模式，Edge 模式是 IE 浏览器最新的渲染引擎，能够更好地支持 HTML5 和 CSS3 等现代 Web 标准。
  • 作用： 确保网页在 IE 浏览器上尽可能地以最佳的方式呈现，避免出现兼容性问题。 避免老旧的IE版本用旧的渲染模式来展示你的网页，导致样式错乱或者功能不正常。

<meta name="viewport" content="width=device-width, initial-scale=1.0">

  • 目的： 针对移动设备优化网页显示。
  • 解释：
    • name="viewport"： 表示设置视口 (viewport)，视口是浏览器中用来显示网页的那部分区域。
    • content="width=device-width, initial-scale=1.0"： 设置视口的具体属性：
      • width=device-width： 让视口的宽度等于设备的宽度。 这意味着网页会根据手机或平板的屏幕宽度自动调整大小。
      • initial-scale=1.0： 设置初始的缩放比例为 1.0，也就是不缩放。 用户可以自由缩放，但默认情况下网页不会被缩小或放大。
  • 作用： 使网页在各种移动设备上都能正确显示，避免出现横向滚动条或者网页被过度缩放的情况。 这对响应式网页设计至关重要。

6.基本的HTML标签

6.1.HTML标题标签

HTML的文章标题标签，如：<h1></h1>~<h6></h6>六个级别，效果依次减少，并且每个标题都是独占一行空间。

注意：没有<h7></h7>标签；

6.2.换行与空格

浏览器总是会截短 HTML 页面中的空格。如果在文本中写 10 个空格，在显示该页面之前，浏览器会删除它们中的 9 个。如需在页面中增加空格的数量，需要使用&nbsp;字符实体。

不间断空格 nbsp 即 non-breaking space，与一般的空格有所区别。也就是说， 用于html代码中用于产生空白，并阻止浏览器在这个空白处换行。nbsp在两个<img>标签中使用 可以防止由于图片太宽超出浏览器宽度而导致的折行。也就是说如果使用一般的空格，当浏览器宽度不够时，会把第二个img折到下一行，而用 这两个img都会显示在同一行。

换行符常用 <br/> 表示。下面是使用换行和空格的简单例子：

6.3.字体、段落、图片、超链接和列表标签

1）字体加粗、倾斜与删除标签分别为：

  • 加粗1<b></b>：加粗文字1
  • 加粗2<strong></strong>：加粗文字2
  • 倾斜1<i></i>：倾斜文字1
  • 倾斜2<em></em>：倾斜文字2
  • 删除1<s></s>：删除文字1
  • 删除2<del></del>：删除文字1

2）段落是通过 <p></p> 标签进行定义的。每个 <p></p> 标签内的内容为一个段落，独占一行。

3）图片标签为<img scr="图片名" alt="加载中" width="" height="" title="这是提示文字">，其中：

  • src: 表示资源,图片加载的名字
  • alt: 图片没有被正常加载时显示
  • width(宽),height(高): 设置图片的像素大小
  • title: 鼠标悬停在图片上的提示文字

4）超链接（点击链接文字跳转链接）的使用标签为 <a href="链接网址">链接文字</a> ；若指定 href 为 # 号，表示空链接，作用为：

  1. 暂时不知道点击之后跳转到哪里，使用空链接占位；
  2. 刷新界面。

5）列表可分为无序列表和有序列表。无序列表的使用标签为：<ul><li></li><li></li><li></li></ul>

注意：

（1）ul标签可以嵌套若干个li标签；

（2）每一个li标签代表着每一条数据；

（3）每个li标签之间没有顺序；

如：

  • python
  • java
  • go

有序列表的使用标签：<ol><li></li><li></li><li></li></ol>

注意：

（1）有序；

（2）且是ol开始的；

如：

  1. 基础班级
  2. 测试
  3. python

span标签：<span></span> 小盒子，一行可以放多个；

6.5.表格标签

表格标签主要由行和列组成：

  • <th> 标签： 表格的表头单元格 (表头，通常是第一行或第一列)。
  • <td> 标签： 表格的数据单元格 (表格中存放数据的普通单元格)。 <td> 标签必须包含在 <table> 标签内。

（1）align / valign：表格水平/垂直对齐方式。如：left，center，right；

（2）border：表格边框。如：1；

（3）cellpadding：单元边沿与其内容之间的空白，默认为1像素；

（3）表格尾部区域：<tfoot> 标签；

（1）跨行合并：rowspan="合并单元格的个数"，最上侧单元格为目标单元格；

6.6.表单标签

表单标签（用于收集用户的输入信息，表示为文档中的一个区域，此区域包含交互控件，将用户收集到的信息发送到 Web 服务器）我们可以直接称为 form 标签，标签书写为：<form action=""></form>。form表单标签里面就是所有用户填写的表单数据；

（1）属性 action：把表单数据交给指定后台程序去处理；

（2）属性 method：传递数据时方式方法。

​ 1.默认为post请求（隐式提交数据）；

​ 2.get明文传送数据；

（3）属性 label：为表单元素添加标签，提高可访问性；

（4）属性 input：<input> 元素是最常用的表单元素之一，它可以创建文本输入框、密码框、单选按钮、复选框等。type 属性定义了输入框的类型，id 属性设置唯一标识符（标识符关联 <label> 标签，<label>标签为表单元素，如复选框，提供描述性的文本标签），主要用于CSS样式设计和JavaScript脚本操作。name 属性用于标识表单字段，后端服务器可以通过该属性获取用户值。

输入框与单选、多选框

输入框（input标签）：<input type="text" placeholder="请输入用户名">

（1）属性 type：“text” 表示普通单行文本框，“password” 表示密码单行输入框；

（2）属性 placeholder：给用户提示，提示用户体验感。

单选、多选框（input标签）：<input type="" name="">

（1）属性 type：“radio” 表示单选框，“checkbox” 表示多选框；

（2）属性 name：选框的名字；

下拉选择框

下拉框（select标签，嵌套若干个option标签，其中每一个option为下拉框中的一个选项）：

（1）属性 value 定义了当用户选择该选项时，表单提交给服务器的值；

（2）属性 selected="selected"，表示默认选中状态。

文本域

文本域（textarea 标签）：<textarea name="", cols="30" rows="10"></textarea>

文本框默认每行宽度大约能显示 30 个字符，高度可显示10行。 注意，这只是一个建议值，实际宽度会受到字体和CSS样式的影响。

按钮标签（普通按钮、重置按钮、提交按钮）

按钮（input标签）：<input type="" value="">

（1）属性 type："button"值表示普通按钮，“reset”值表示重置按钮（该按钮自动会有重置的文字），“submit”值表示提交按钮（该按钮自动会有提交的文字）；

（2）属性 value：按钮上显示的文字内容
