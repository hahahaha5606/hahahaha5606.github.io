一、简介

Redis（Remote dictionary server）是一个开源的基于内存的数据存储系统，可用于数据库、缓存和消息队列等各种场景。在早期的 MySQL 应用中，由于磁盘的IO读写速度与内存相比太慢，难以满足互联网时代海量数据的需求。而 Redis 作为一种基于内存的数据存储系统， 能把数据存储在内存中，大大缓解了 MySQL 的应用瓶颈。

Redis 支持以下五种基本数据类型和五种高级数据类型，分别为：

Redis 的使用方式主要有三种：

  • 命令行接口 CLI（Command Line Interface）
  • 应用程序接口 API（Application Programming Interface）：使用 Python 、Java 等常见编程语言使用 Redis
  • 图形化用户接口 GUI（Graphical User Interface）：如 RedisInsight 等

Redis 有以下优点：

  • 性能极高，和其他数据库相比差了一个数量级
  • 数据类型丰富，单键值对最大支持512M大小的数据
  • 简单易用，支持所有主流编程语言
  • 支持数据持久化、主从复制、哨兵模式等高可用特性
